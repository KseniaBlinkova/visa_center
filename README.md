# Лабораторная работа №4: Разработка REST API на Node.js

## Описание
Разработка серверной части приложения для управления данными каталога визового центра.

## Функционал API (CRUD)
Сервер поддерживает следующие операции:
- **GET /stocks** — Получение списка всех виз.
- **POST /stocks** — Добавление новой услуги в каталог.
- **PATCH /stocks/:id** — Редактирование информации о визе.
- **DELETE /stocks/:id** — Удаление услуги из базы.

## Стек
- Node.js, Express.js.
- Тестирование: Postman.
