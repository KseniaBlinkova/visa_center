## Лабораторная работа №6 (Async/Await, Fetch, Vite)

**Цель:**  
Заменить механизм XHR на современный fetch с async/await и объединить клиентскую и серверную части с помощью сборщика Vite.

**Задачи:**
- Переписать все запросы к API с XHR на fetch.
- Использовать синтаксис async/await для обработки асинхронных операций.
- Настроить сборку фронтенда с помощью Vite.
- Объединить клиентскую и серверную части в единую структуру.
- Сохранить полную функциональность CRUD-операций для управления визовыми записями.
