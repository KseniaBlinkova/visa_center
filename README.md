**Цель:**  
Добавить калькулятору интерактивность с помощью JavaScript и реализовать тематическую функцию для предметной области «Визовый центр».

**Задачи:**
- Реализовать базовые арифметические операции.
- Разработать авторскую функцию `calculateVisaEquivalent` для конвертации суммы в валюты.
- Реализовать механизм динамической смены тем оформления.
- Адаптировать названия функций под предметную область (например, `addApplications`, `multiplyIssuedVisas`).
