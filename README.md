## Домашнее задание

**Цель:**  
Интегрировать в проект авторские алгоритмы обработки данных и 3D-модель с использованием библиотеки Three.js.

**Задачи:**
- Реализовать алгоритм поиска максимальной последовательности единиц (задача 2.3) для анализа истории одобрений виз.
- Реализовать функцию `merge` (задача 3.1) для объединения объектов с общими и дополнительными документами.
- Подключить библиотеку Three.js и GLTFLoader через importmap.
- Добавить на страницу детального просмотра 3D-модель оборудования с возможностью изменения масштаба и положения.
