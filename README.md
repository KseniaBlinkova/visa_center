## Лабораторная работа №3 (Простое веб-приложение. Верстка)

**Цель:**  
Познакомиться с Node.js и npm, разработать приложение-галерею с каталогом видов виз.

**Задачи:**
- Создать главную страницу с карточками видов виз, поиском и фильтрацией.
- Реализовать страницу детального просмотра информации о визе.
- Добавить кнопки добавления (копирование первой карточки) и удаления карточки.
- Реализовать навигацию с кнопкой «Домой» в хедере обеих страниц.
