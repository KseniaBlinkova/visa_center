# Лабораторная работа №3: Проектирование интерфейса «Визовый центр»

## Описание
Перевод статического приложения на компонентный подход. Реализовано двухстраничное веб-приложение (каталог услуг и страница деталей).

## Реализованный функционал
- **Компонентная база:** Созданы классы `VisaCardComponent` и `MainPage` для динамической отрисовки интерфейса.
- **Инструментарий:** Инициализация проекта через `npm`, управление стилями с помощью Bootstrap 5.
- **Интерактив:** Реализован поиск (фильтрация) по названию и возможность добавления/удаления карточек.
