## Лабораторная работа №1 (Калькулятор HTML/CSS)

**Цель:**  
Создать калькулятор, стилизованный под официальный сайт единого визового центра, и освоить основы HTML и CSS.

**Задачи:**
- Разработать интерфейс калькулятора с полем вывода результата, кнопками чисел и операций.
- Выполнить верстку с использованием цветовой гаммы сайта-референса (синий, красный, белый, серый).
- Реализовать хедер в стиле визового центра.
- Добавить кнопку перехода в GitHub-репозиторий.
- Добавить блок с информацией об авторе.
