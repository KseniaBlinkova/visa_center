## Лабораторная работа №5 

**Цель:**  
Интегрировать созданный API в клиентскую часть и реализовать полноценное взаимодействие с сервером с использованием XMLHttpRequest.

**Задачи:**
- Подключить фронтенд из лабораторной работы №3 к бэкенду из лабораторной работы №4.
- Реализовать методы работы с API через XHR.
- Добавить страницу редактирования визовых карточек.
- Реализовать загрузку, добавление, редактирование и удаление данных без перезагрузки страницы.
- Настроить обход CORS с помощью браузерного расширения CORS Unblock.
