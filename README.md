## Лабораторная работа №4 (Реализация API на Node.js)

**Цель:**  
Разработать REST API для управления каталогом визовых услуг с поддержкой CRUD-операций.

**Задачи:**
- Реализовать сервер на Node.js с использованием Express.js.
- Организовать хранение данных в локальном JSON-файле.
- Реализовать эндпоинты:
  - `GET /visas` — получение списка (с фильтрацией по title)
  - `GET /visas/:id` — получение одной записи
  - `POST /visas` — создание новой записи
  - `PATCH /visas/:id` — редактирование записи
  - `DELETE /visas/:id` — удаление записи
- Протестировать все методы через Postman.
